Detailed Description

Partial implementation of std::dequeue. Just providing the functionality that's needed to keep a record of previous neural network results over a short time period, so they can be averaged together to produce a more accurate overall prediction. This doesn't use any dynamic memory allocation so it's a better fit for microcontroller applications, but this does mean there are hard limits on the number of results it can store.
